# 🚀 Campus Feud 🎮

A real-time, multi-device **Family Feud–style game system** built for college events, hackathons, and live competitions.
Designed to run seamlessly across **Display, Judge, and Leaderboard devices** with synchronized gameplay and cinematic UI.

---

## Sample Deplued URLs - 
  * Main Projection Site - https://campus-feud.vercel.app/
  * Judge Portal - https://campus-feud.vercel.app/judge
  * Leaderboard (Projection) - https://campus-feud.vercel.app/leaderboard
  * Leaderboard Edit - https://campus-feud.vercel.app/leaderboard/edit


## ✨ Features

* 🎯 **Multi-Device Sync (Firebase)**

  * Control game from mobile (Judge)
  * Display on projector/screen
  * Edit leaderboard from tablet/laptop

* 🧠 **Dynamic Question System**

  * 35 + 3 + 1 round structure
  * Custom answer points (editable)
  * Sequential reveal with animations

* 🔊 **Game Show Experience**

  * Answer reveal sounds
  * Strike buzzer
  * “Be Specific” overlay
  * Timer with end alarm

* 🎮 **Judge Control Panel**

  * Round-wise navigation
  * Question preview on hover
  * One-click reveal system
  * Strike + special triggers

* 📊 **Leaderboard (Animated)**

  * Live score updates
  * Ranking animations
  * Bar chart UI with glow effects

* ⏱ **Cinematic Timer**

  * Start / Stop / Reset
  * Auto disappear after time ends
  * Visual + audio alerts

---

## 🏗️ Tech Stack

* **Frontend:** React + TypeScript
* **State Management:** Zustand
* **Realtime Sync:** Firebase Realtime Database
* **Deployment:** Vercel
* **Styling:** Custom CSS (Game-show UI)
